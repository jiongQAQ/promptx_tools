# PromptX Workspace

这是 promptx_tools 项目的 PromptX 工作空间。

## 目录结构

- `resource/` - PromptX 资源文件目录
  - 角色定义
  - 执行模式
  - 知识库
  - 思维模式

## 使用方法

配置完成后，重启 Claude Code 会话，然后可以使用以下 PromptX 工具：

- `mcp__promptx__promptx_init` - 初始化 PromptX
- `mcp__promptx__promptx_action` - 执行 PromptX 动作

## 扩展

可以在 `resource/` 目录下添加更多的 PromptX 资源文件来扩展功能。
